# Проявка — спецификация

*2026-06-10. Дизайн утверждён через judge-panel (3 направления × 3 судьи), победитель — «галерея» с украденными идеями остальных.*

## Что это

«Проявка» — улучшайзер фотографий: загружаешь фото, нейросеть Real-ESRGAN
увеличивает его в 2×/4× и вытягивает детали. Делается как подарок-сервис для
одного человека (не-программист, Windows), но выглядит и устроено как
настоящий open-source продукт.

Имя — метафора проявки плёнки: фото «проявляется» из размытия, и это же —
визуальный язык прогресса.

## Форм-факторы

1. **Web** — FastAPI на сервере Севы (LXC blueprint-ai), она открывает ссылку.
2. **Desktop** — Tauri v2 (Windows `.exe` + macOS `.dmg`), тот же UI из
   `static/`, движок — локальный sidecar `realesrgan-ncnn-vulkan` (MIT).
   Сборки делает GitHub Actions.

## Архитектура (web)

```
браузер ──POST /api/jobs──▶ FastAPI ──queue.Queue──▶ worker-поток
   ▲                          │                         │
   └─poll GET /api/jobs/{id}──┘                  Engine.upscale()
                                                  ├─ torch (CPU, сейчас)
                                                  ├─ ncnn  (GPU, после reclaim)
                                                  └─ fake  (тесты/дев)
```

- Очередь в памяти, один worker (фото обрабатываются по одной).
- Джобы на диске `data/jobs/<id>/`, TTL 24 ч, чистильщик-поток.
- Движок выбирается автоматом: есть `/dev/dri` и бинарник → ncnn (GPU),
  иначе torch CPU.

### Модели

| Ключ | Файл | Роль |
|---|---|---|
| `soft` | realesr-general-x4v3 (компактная) | дефолт: быстрая, мягкая |
| `detail` | RealESRGAN_x4plus | «попробовать по-другому»: максимум деталей |
| `art` | RealESRGAN_x4plus_anime_6B | рисунки/иллюстрации |

2× = апскейл 4× + даунскейл Lanczos (модели нативно только 4×).

### Лимиты

- Файл ≤ 30 МБ; вход ≥ 4.6 Мп ужимается (клиент и так шлёт ≤ 2048px).
- Очередь ≤ 6 задач → 429 с человеческим текстом.
- EXIF-поворот применяется на входе; PNG остаётся PNG, остальное → JPEG q95.

## UI (общий для web и desktop)

Один экран, три состояния в одной «раме» (фикс. aspect-ratio — ноль прыжков):

1. **idle** — дропзона «Брось фото сюда или нажми, чтобы выбрать».
2. **processing** — превью фото проявляется из blur по мере прогресса (rAF,
   не CSS-transition); тонкая линия-прогресс 2px; фразы каждые 9 с
   («Вглядываюсь в детали…»), после 60 с — фраза терпения; wakeLock.
3. **done** — before/after слайдер (clip-path + pointer events,
   touch-action: none на всей раме), авто-проезд 15→55% обучает без слов;
   «Скачать в полном размере» (sticky, navigator.share на тач) →
   «Попробовать по-другому» (цикл моделей) → «загрузить другое фото».

Стиль: фон #FAF9F6, текст #1B1A17, акцент терракота #A35A3C, hairline
#E9E6DF; вордмарк Spectral 300, остальное системный шрифт; «сделано Севой ·
с любовью» в футере. Прогресс честный: визуально асимптотический, 100% —
только по реальному ответу сервера.

Клиент перед отправкой ужимает фото до 2048px по длинной стороне — главная
защита от CPU-таймаута. PNG ужимается в PNG (важно для рисунков и
прозрачности), остальное — в JPEG 0.9 на белом фоне. HEIC, который браузер не
смог раскодировать, уходит как есть — сервер открывает через pillow-heif.

## Инфраструктура

- **Сейчас**: blueprint-ai (LXC 105, 4 vCPU, 8 ГБ), systemd-сервис, порт 8000,
  ENGINE=auto → torch CPU.
- **GPU-upgrade** (подготовлено, НЕ выполнено): RX 7700/7800 XT сейчас на
  vfio-pci (winvm остановлена). `/root/gpu-upscaler/reclaim-gpu.sh` на
  Hosting400 возвращает карту хосту без ребута, `lxc-gpu.conf` — две строки
  для проброса /dev/dri в CT 105. После этого ENGINE=auto сам перейдёт на
  ncnn/Vulkan (секунды вместо минут). Обратно — `release-gpu.sh`.
- **Публичный доступ** — открытый вопрос для Севы: tailscale funnel /
  cloudflared (нужен его аккаунт) / reverse-tunnel на bfl-prod.

## Тесты

- pytest: API на fake-движке (10 тестов: happy path, PNG/JPEG, scale,
  EXIF-поворот, лимиты, мусор, 404).
- Playwright: e2e сценарий на fake-движке + скриншоты desktop/mobile;
  прод-смоук на реальном движке.
- Ревью: агенты по линзам (UI-скриншоты, код, качество апскейла на реальных
  фото) с адверсариальной проверкой.
